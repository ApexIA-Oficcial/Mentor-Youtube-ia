Política de Privacidade ApexiA

Última atualização: 7 de Fevereiro de 2026

1. Dados acessados

A aplicação ApexiA Mentor Supremo acede, através das APIs oficiais do Google, aos seguintes dados do seu canal de YouTube: nome do canal, contagem de subscritos, contagem de vídeos e métricas de desempenho (visualizações, CTR e retenção).

2. Utilização de dados

Estes dados são utilizados exclusivamente para que a nossa Inteligência Artificial (Google Gemini) analise a performance do seu canal e forneça recomendações estratégicas de crescimento. Os dados servem apenas como contexto para a mentoria.

3. Compartilhamento de dados

Nós não compartilhamos os seus dados com terceiros. Os dados capturados são processados em tempo real e enviados apenas para a API oficial do Google Gemini para a geração das respostas da IA.

4. Armazenamento e proteção de dados

A ApexiA não possui base de dados externa. Todas as informações são processadas "client-side" (no seu navegador). O histórico de conversas e as métricas capturadas são armazenados apenas localmente no localStorage do seu navegador, garantindo que só você tenha acesso a eles.

5. Retenção e Exclusão de dados

Os dados são mantidos apenas enquanto a sua sessão estiver ativa. Pode excluir permanentemente todos os dados e o histórico de conversas a qualquer momento clicando no botão "Sair" ou "Limpar Chat" dentro da aplicação.
